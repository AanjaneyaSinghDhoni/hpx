Struct template is_timed_executor

hpx::parallel::v3::is_timed_executor

Synopsis

// In header: <hpx/parallel/executors/timed_executor_traits.hpp>

template<typename T> 
struct is_timed_executor {
};

Description

  1. The type is_timed_executor can be used to detect executor types for the purpose of excluding function signatures from otherwise ambiguous overload resolution participation.

  2. If T is the type of a standard or implementation-defined executor, is_timed_executor<T> shall be publicly derived from integral_constant<bool, true>, otherwise from integral_constant<bool, false>.

  3. The behavior of a program that adds specializations for is_timed_executor is undefined.
